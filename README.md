📌 Site LabI – Atividade jQuery

Projeto desenvolvido para a disciplina Tecnologias para Interfaces de Aplicações Web do curso de Análise e Desenvolvimento de Sistemas – IFTM.

O objetivo da atividade é praticar seletores e eventos utilizando jQuery, criando uma página interativa que apresenta informações sobre o Laboratório de Ideação (LabI) e a cultura maker.

🧠 Objetivo da Atividade

Desenvolver uma página web com um menu de navegação onde cada item:

Exibe uma seção específica da página

Esconde as outras seções

Altera o tema de cores do layout

As seções implementadas são:

Sobre

Equipe Gestora

Agenda

Projetos

Contato

Cada seção possui um tema de cor diferente aplicado com jQuery.

🛠 Tecnologias Utilizadas

HTML5 – Estrutura da página

CSS3 – Estilização e layout

JavaScript

jQuery – Manipulação do DOM, eventos e seletores
